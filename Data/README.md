# Data Directory

This directory contains galaxy property datasets from the Portsmouth Group spectro-photometric model fitting catalog, organized by galaxy classification type.

## Dataset Files

### 1. Random-AGN-Galaxies-PortsmouthGroup_properties.csv
Galaxy properties for **Active Galactic Nuclei (AGN)** galaxies derived from the spectro-photometric model fitting catalog. AGN are galaxies with supermassive black holes at their centers that actively accrete matter and emit significant energy.

**File size:** ~114 KB

### 2. Random-SB-Galaxies-PortsmouthGroup_properties.csv
Galaxy properties for **Starburst (SB)** galaxies derived from the spectro-photometric model fitting catalog. These are galaxies undergoing an exceptionally high rate of star formation.

**File size:** ~120 KB

### 3. Random-SF-Galaxies-PortsmouthGroup_properties.csv
Galaxy properties for **Star-Forming (SF)** galaxies derived from the spectro-photometric model fitting catalog. These are normal galaxies with moderate ongoing star formation rates.

**File size:** ~123 KB

## Data Format

All datasets are stored in **CSV (Comma-Separated Values)** format, making them easily readable by:
- Python (using pandas, csv module, etc.)
- Excel and other spreadsheet applications
- Text editors

## Usage

These datasets are designed to be used with Python scripts for exploratory data analysis and comparison of galaxy properties across different galaxy types. Refer to the main repository scripts for examples of loading and analyzing these datasets.

## Data Source

These datasets are from the **Portsmouth Group spectro-photometric model fitting catalog**, containing observational properties derived from spectral and photometric modeling of randomly selected galaxies from different classification categories.
