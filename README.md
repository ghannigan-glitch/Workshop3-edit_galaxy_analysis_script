# Workshop3-edit_galaxy_analysis_script

## Description
This repository is for practicing a GitHub version control workflow on a python script for reading in and exploring a table of galaxy properties.

## Installation Instructions/ Required Packages 
This code requires python 3.xxx meaning python 3 or later 
This code requires the following packages: 
pandas, matplotlib, numpy, astropy.units, and FlatLambdaCDM from astropy.cosmology

## Usage
There are no other programs that are required to install for this code however there is data that needs to be downloaded including the following csv:
'Data/Random-SF-Galaxies-PortsmouthGroup_properties.csv', 'Data/Random-SB-Galaxies-PortsmouthGroup_properties.csv', and 'Data/Random-AGN-Galaxies-PortsmouthGroup_properties.csv'
